Welcome to my Spring Demo!

Relevant endpoints:

Endpoint Description
/api/stores REST endpoint to retrieve all stores
/api/store/1 REST endpoint to retrieve a single store by id
/api/products REST endpoint to retrieve all products
/api/product/1 REST endpoint to retrieve a single product by id
/api/product/1/stock REST endpoint to retrieve the stock for a single product for all stores
/api/store/1/stock REST endpoint to retrieve the stock for a single store for all products
/api/store/1/product/1/stock REST endpoint to retrieve the stock for a single store and a single product
/api/reservations/1 REST endpoint to retrieve a single reservation
/api/store/1/reservations REST endpoint to retrieve all reservations for a single store
/api/store/1/product/1/reservations REST endpoint to retrieve all reservations for a single product in a single store
/api/h2-console Embedded GUI console for H2 database (password = "password")
